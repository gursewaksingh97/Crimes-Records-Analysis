# 🚔 Crimes Records Analysis

A modern data analytics project focused on identifying criminal trends, demographic patterns, behavioral indicators, and operational insights using **Power BI**, **Excel**, and data storytelling techniques.

---

# 📌 Project Overview

This project analyzes criminal incident records to uncover meaningful insights related to:

* Crime distribution
* Age group involvement
* Weapon usage
* Mental illness indicators
* Threat levels
* Regional crime intensity
* Operational transparency using body camera data

The dashboard and analysis help in understanding criminal behavior patterns and support data-driven decision-making.

---

# 🎯 Problem Statement

The objective of this analysis is to identify patterns, trends, and risk indicators within criminal encounter records using demographic, behavioral, and operational data.

The analysis aims to help understand factors such as:

* Threat levels
* Age distribution
* Gender & race analysis
* Mental illness indicators
* Operational response patterns

for better analytical insights and visualization.

---
# 📂 Dataset Features

The dataset includes:

* Incident ID
* Name
* Date
* Manner Of Death
* Armed Type
* Age
* Gender
* Race
* City
* State
* Threat Level
* Flee Status
* Body Camera Availability
* Mental Illness Indicators
